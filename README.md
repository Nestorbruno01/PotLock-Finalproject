## Final submission for Tech Basics II

## 🔐 PotLock
PotLock is a tiny DIY lock powered by an Arduino. <br>
Turn the dial, press the button, and hope you remember the code. <br>
If you get it right the servo unlocks. <br>
If not… try again 😉<br>
With a screen, lights, sound, and a moving lock, PotLock turns a simple box into a little security system.
